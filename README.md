# SFIT Execute API Service

This project is a Spring Boot based execution service that dynamically clones test repositories (Jest, JUnit, TestNG, Playwright, Robot Framework), installs dependencies, and executes test cases.

## Tech Stack

- Java 17
- Spring Boot
- Docker
- Node.js
- Playwright
- Jest
- JUnit
- TestNG
- Git

---

## Prerequisites

Make sure the following are installed:

- Java 17
- Maven or Gradle
- Docker
- Git
- Node.js (if running locally)

---
